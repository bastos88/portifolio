# 🎮 Leonardo Bastos — Arcade Developer Portfolio

Portfólio pessoal desenvolvido com uma estética inspirada em jogos arcade, pixel art e interfaces retro. O projeto apresenta a minha trajetória, competências técnicas, projetos e formas de contacto através de uma experiência visual interativa e responsiva.

## 📌 Sobre o projeto

Este portfólio foi criado para unir desenvolvimento web moderno com uma identidade visual inspirada em videojogos clássicos.

A navegação foi pensada como uma jornada de progresso, onde cada section representa uma fase da minha evolução profissional:

* `INSERT COIN` — apresentação inicial
* `PLAYER PROFILE` — perfil profissional
* `SKILLS UNLOCKED` — competências técnicas
* `SELECT PROJECT` — projetos desenvolvidos
* `SOURCE CODE` — tecnologias e abordagem de desenvolvimento
* `GAME PROGRESS` — linha do tempo profissional
* `CONTACT TERMINAL` — contactos e redes profissionais

## ✨ Funcionalidades

* Layout responsivo para desktop, tablet e mobile
* Navbar adaptável conforme o scroll
* Navegação suave entre sections
* Timeline profissional estilo arcade
* Cards de projetos com estética pixel-art
* Componentes reutilizáveis em React
* Botão flutuante de retorno ao topo com foguete animado
* Integração com links de GitHub, LinkedIn e currículo
* Animações leves e transições visuais
* Interface inspirada em retro gaming, CRT e HUDs de videojogos
* Estrutura preparada para acessibilidade e navegação por teclado

## 🚀 Como executar localmente

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

Entre na pasta do projeto:

```bash
cd SEU-REPOSITORIO
```

Instale as dependências:

```bash
npm install
```

Execute o projeto em ambiente de desenvolvimento:

```bash
npm run dev
```

Depois, abra o endereço apresentado no terminal, normalmente:

```txt
http://localhost:5173
```

## 📦 Scripts disponíveis

```bash
npm run dev
```

Inicia o servidor de desenvolvimento.

```bash
npm run build
```

Gera a versão de produção do projeto.

```bash
npm run preview
```

Executa localmente a versão gerada para produção.

```bash
npm run lint
```

Verifica possíveis problemas de qualidade e padrão de código.

## 🎨 Conceito visual

O design utiliza uma combinação de:

* Pixel art
* Interface arcade
* Tipografia inspirada em jogos retro
* Tons bege, verde arcade, amarelo e preto
* Elementos visuais inspirados em HUDs, menus de videojogos e terminais antigos
* Bordas pixeladas e efeitos de progressão de jogo

A proposta é transformar um portfólio técnico em uma experiência visual memorável, sem perder clareza, legibilidade ou foco profissional.
